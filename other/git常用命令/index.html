git常用命令

发表于 2017-02-11   |   分类于 Other

常用命令

  • git init 初始化仓库
  • git clone url 检出
  • git add 添加
  • git add . 或者 git add -A 添加当前目录下所有文件
  • git commit -m “comment” 文件提交
  • git commit –amend 与上次commit合并
  • git push origin master 推送至origin的master分支
  • git pull 更新并合并
  • git fetch 更新
  • git log 查看提交日志
  • git diff 查看修改的内容
  • git diff HEAD
  • git diff
  • git reset 回滚到上次提交
  • git reset 回滚到上次提交
  • git reset –hard
  • git checkout
  • git checkout – files
  • git checkout HEAD – files
  • git checkout -b branch_name 创建branch_name分支并切换
  • git checkout master 切换到master
  • git branch -d branch_name 删除branch_name分支
  • git push origin branch_name 推送branch_name到origin
  • git merge branch_name 合并branch_name到当前分支
  • git rebase
