发表于 2017-07-26   |   分类于 Android

Property Animation 属性动画

属性动画与View动画的比较

  • view动画只能为view添加动画;
  • view动画支持的种类少;
  • viwe动画只改变绘制效果，不改变view的属性
  • view动画代码量少，使用简单
  • 属性动画可以为任意对象的任意属性添加动画
  • 属性动画会改变对象的属性
  • 当对象属性变化时，属性动画会自动刷新屏幕

Tween animation

Tween Resource文档
通过Animation在单个图像上执行一系列变换来产生动画效果

语法

元素

  • set 包含其它动画元素或其它容器的容器 相当与AnimationSet
    • android:interpolator Interpolator resource 设置插值器
    • android:shareInterpolator Boolean 如果为true则让所有子元素共享插值器
  • 通用属性 继承Animation的所有动画类都能使用
    • android:detachWallpaper
    • android:duration
    • android:fillAfter
    • android:fillBefore
    • android:fillEnable
    • android:interpolator
    • android:repeatCount
    • android:repeatMode
    • android:startOffset
    • android:zAdjustment
  • alpha 淡入淡出动画 相当与AlphaAnimation
    • android:fromAlpha Float 开始时的透明度 0.0表示完全透明 1.0 表示完全不透明
    • android:toAlpha Float 结束时的透明度
  • scale 改变大小动画 通过设置pivotX和pivotY来指定放大或缩小动画的中心点 相当与ScaleAnimation
    • android:fromXScale Float 开始时的X轴大小偏移 1.0表示没有变化
    • android:toXScale Float 结束时的X轴大小偏移
    • android:fromYScale 同上
    • android:toYScale 同上
    • android:privoX 中心点X坐标
    • android:privoY 中心点Y坐标
  • translate 垂直或水平动画 相当与TranslateAnimation
    • 下列属性值有三种格式 值在-100~100间并且后缀为% 表示相对与自身的百分百 后缀为%p 表示相对与parent的百分百 没有后缀 表示一个绝对值
    • android:fromXDelta
    • android:toXDelta
    • android:fromYDelta
    • android:toYDelta
  • rotate 旋转动画 相当与RotateAnimation
    • android:fromDegrees
    • android:toDegrees
    • android:pivotX
    • android:pivotY

Frame animation 帧动画

通过AnimationDrawable显示一系列图片来产生动画
相对与Tween animation 的xml定义在res/anim/filename.xml, Frame animation的xml定义在res/drawable/filename.xml

语法

元素

  • animation-list 必要的元素而且必须是根元素 包含一个或多个item元素
    • android:oneshot Boolean true则只进行一次动画 false则循环
  • item 表示动画的一帧 必须为animation-list 的子元素
    • android:drawable Drawable resource drawable资源
    • android:duration Integer 一帧的时长

Transition动画

文档
例子

概念

  • Scene 保存ViewGroup所有元素的状态，包括View和属性
  • Target 在Scene里的View对象，可以在Scene移除或添加Target
  • Transition Scene切换所用的动画信息对象
  • TransitionSet 动画集合
  • TransistionManager 在Scene间施加相应的Transition动画
  • TransitionManager.beginDelayedTransition()
  • TransistionListener 类似Transistion的生命周期回调

Activity和Fragment的Transition

  • exit transition: A start B, A中View的动画
  • enter transition: A start B, B中View的动画
  • return transition: B return A, B中View的动画
  • reenter transition: B return A, A中View的动画
    Fragment： added, removed, attached, detached, shown

    非享元切换Transition

  • setExitTransition() setEnterTransition()
  • setReturnTransition() setReenterTransition()
  • ViewGroup.setTransitionGroup(true)

    享元切换Transition

  • setSharedElementEnterTransition()
  • setSharedElementReturnTransition()

Transition动画注意事项

  • 在SurfaceView上可能表现不正常
  • 在TextureView上施加某些Transition效果，可能达不到预期效果
  • 继承自AdapterView的View，比如ListView上可能会出错
  • 对TextView的size施加Transition效果时，文字的位置会突变
  • 不要通过scene的actions在scene间传数据，用callback

Interpolators 插值器

一些已经实现的插值器

java类 xml id 描述
AccelerateDecelerateInterpolator @android:anim/accelerate_decelerate_interpolator 动画始末速率较慢，中间加速
AccelerateInterpolator @android:anim/accelerate_interpolator 动画开始速率较慢，之后慢慢加速
AnticipateInterpolator @android:anim/anticipate_interpolator 开始的时候从后向前甩
AnticipateOvershootInterpolator @android:anim/anticipate_overshoot_interpolator 类似上面AnticipateInterpolator
BouncelInterpolator @android:anim/bounce_interpolator 动画结束时弹起
CycleInterpolator @android:anim/cycle_interpolator 循环播放速率改变为正弦曲线
DecelerateInterpolator @android:anim/decelerate_interpolator 动画开始块然后慢
LinearInterpolator @android:anim/linear_interpolator 动画匀速改变
OvershootInterpolator @android:anim/overshoot_interpolator 向前弹出一定值之后回到原来位置
PathInterpolator 新增，定义路径坐标后按照路径坐标来跑
